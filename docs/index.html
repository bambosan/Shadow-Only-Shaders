Shadow Only Shaders

About

This is a shaderpack for Minecraft: Java Edition with Optifine mod which will add dynamic shadows with some configurable features.
The shaders is very minimalist because it is designed for gameplay. In other words, it is a survival-friendly shader. So if you are bored with Minecraft graphics that are too flat, try this shader.

Feature

  • Percentage Closer Filtering (PCF) shadows / just shadow with smooth filter
  • Percentage Closer Soft Shadows (PCSS) / shadow with penumbra and smooth filter
  • Custom fog and synchronized it with the sky

Warning

  • This shaderpack only works for Minecraft: Java Edition 1.14-1.19 (stable version) with Optifine mod
